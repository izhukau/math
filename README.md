# Mathematics

A collection of mathematical proofs, research, and computational experiments.

## Contents

### Proofs

- **[Inverse Function Theorem](proofs/inverse-function-theorem/)** — Full proof of the multivariable Inverse Function Theorem, including local invertibility (via Banach Fixed-Point Theorem), differentiability of the inverse, and the inverse Jacobian formula.

### Optimization

- **[Gradient Descent](optimization/gradient-descent/)** — Theory and convergence analysis of gradient descent variants (constant step, Armijo backtracking, steepest descent with exact line search). Includes computational experiments on quadratic and Rosenbrock functions.
